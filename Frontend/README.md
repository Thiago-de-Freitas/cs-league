# Gamers League — Frontend

Interface web da plataforma **Gamers League**: ligas, times, partidas e análise de demos (CS2 e outros jogos no futuro). Angular 19.

## Desenvolvimento

```bash
npm install
npm start
```

Acesse http://localhost:4200 — o proxy encaminha `/api` para o backend local.

## Build

```bash
npm run build
```

Saída em `dist/gamers-league/browser`.
